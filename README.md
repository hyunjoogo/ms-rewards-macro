# MS-REWARDS-MACRO
***
[Microsoft Rewards](https://rewards.bing.com/)에서 포인트를 손쉽게 얻는 매크로입니다.

URL : https://ms-reward-macro.netlify.app/

## Usage
***

- 검색할 횟수, 검색어를 입력해주세요
- 검색버튼을 누르면 3초 뒤에 새창으로 검색을 시작합니다.
- 새 창이 3초 뒤에 자동적으로 닫힙니다.
- 지정한 횟수에 맞게 반복합니다.
- 지정 횟수가 완료되면 멈춥니다.

![](./img/flow.gif)

- 중지버튼을 누르면 검색을 중지할 수 있습니다.
